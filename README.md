# TYPO3 extension `ll_anthology`

A TYPO3 extension that provides a generic content display system for listing and viewing records from any configured repository with advanced filtering capabilities.

|-|-----|

| **TER:** | https://extensions.typo3.org/package/liquidlight/typo3-anthology |

## Preconfigured modules

Liquid Light offers several pre-configured modules built with Anthology. If you are interested in listing, filtering and viewing records such as News, Careers, Events, People, Locations etc. then please [get in touch](mailto:info@liquidlight.co.uk).
